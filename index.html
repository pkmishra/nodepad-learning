About.

This is my endeavor to learn node.js.I have followed the tutorial presented on dailyjs (http://dailyjs.com/2010/11/01/node-tutorial/). It is a great starting point and I really thank Alex for writing this. however since the original series was started as late as 2010 some of the libraries and source code(https://github.com/alexyoung/nodepad) is a little bit outdated. I tried to correct the code after each tutorial so that others may use it.
